# Workflow: MEMORY SYNC & HANDOVER

Questo workflow è MANDATORIO all'inizio di ogni sessione o cambio di agente.

## Passaggi:

1.  **Lettura Orientamento**: Leggi `MISSION_CONTROL.md` per comprendere l'obiettivo finale e lo stato del progetto.
2.  **Analisi Errori**: Leggi `DEVELOPER_LOG.md` per non ripetere errori tecnici già risolti.
3.  **Stato Operativo**: Leggi `task.md` per i task in corso.
4.  **Recupero Backlog**: Leggi `BACKLOG.md` per le richieste posticipate.
5.  **Verifica**: Conferma all'utente di aver assimilato questi documenti prima di proporre qualsiasi modifica.
